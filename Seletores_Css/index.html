Adicionando Css ao documento

Temos 3 formas de adicionar um estilo ao documento

      Css Interno

    Utilizado por meio de uma tag

    <style> div { color: red; } </style>

      CSS em um seletor

    <div style="color: blue"><div>



      Css Externo

    Utilizado por meio de um link para uma folha de estilo externa
    com um: ><link rel="stylesheet" href="style.css"> colocado dentro da tag head.


Para uma melhor flexibilidade e facilidade de manutenção,
é indicado o uso da folha de estilo, por link para um documento externo.



Seletor por classe


Utiliza-se o "." para chamar um seletor de classe,
para gerar o estilo daquela classe que ela está atribuída.

<div class="exemplo">Parágrafo>
<div class="exemplo">Div<div>

.exemplo {
color: blue;
}
