Adicionando Css ao documento

Temos 3 formas de adicionar um estilo ao documento

      Css Interno

    Utilizado por meio de uma tag <style> div { color: red; } </style>

      Ou dentro de um seletor


      Css Externo

    Utilizado por meio de um link para uma folha de estilo externa
    com um: <link rel="stylesheet" href="style.css"> colocado dentro da tag head.
